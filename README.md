# Speech-Recognition
DSP Project

The program takes voice commands from the user and executes them.

Available commands:
    - "Hello"
    - "What's up?"
    - "Tell me a joke"
    - "Open website {website name}"
    - "Open program {program name}"
    - "Close program {program name}"
    - "Open file {file name}" 
    - "Goodbye"
